The SIST-02 Citation Style for Multilingual Zotero

This page is a companion to a fully automated version of the SIST-02 citation style (pronounced "shisto-2") maintained by the Japan Science and Technology Agency. This style is intended mainly for use in technical writing.

The examples below were formatted automatically using Multilingual Zotero (MLZ), a variant of official Zotero with support for legal and multilingual referencing. The [src] link next to each entry displays the source from which the citation was generated; the [SIST02] link displays the original citation in the style manual.

Both MLZ and the SIST-02 style are free for immediate use. To get started with the system, in the Firefox browser, install Multilingual Zotero, the Abbreviation Filter and the SIST-02 style from the CitationStylist website. The MLZ manual is available in printed form from Amazon and as a downloadable PDF.

Note that the SIST-02 style makes use of multilingual features of CSL-m, an extended version of the official CSL language, and cannot be used with monolingual reference managers such as Zotero or Mendeley.

If you have any questions or comments, feel free to contact me via Twitter (as @fgbjr), or through the SIST-02 GitHub issue tracker (in English or Japanese).

Enjoy!

「SIST-02」注釈の自動化

このページは、日本科学技術振興機構によって制定および改訂されてきたSIST-02引用スタイル（ "shisto-2"と発音）を自動的に生成するための手引きです。 このスタイルは、主に技術的な文書での使用を意図しています。

以下の例は、公式Zoteroの異版で、法文献や多言語の参照文献形式をサポートするMultilingual Zotero（MLZ）を利用して自動的にフォーマットされたものです。各エントリの横にある[src]リンクは引用が生成されたソースへ、[SIST02]リンクはSIST-02の各解説へ、それぞれリンクしています。

MLZとSIST-02スタイルはどちらもすぐに無料で利用できます。使い始めるには、Firefoxブラウザーを開いて、CitationStylistウェブサイトから、Multilingual Zotero、Abbreviation Filter、SIST-02のスタイルをインストールしてください。MLZのマニュアルは、Amazonから入手、もしくはPDFでダウンロード可能です。

SIST-02スタイルは、公式CSL言語の拡張版であるCSL-mの多言語機能を利用していますので、ZoteroやMendeleyなどの単一言語にしか対応していない文献管理ツールで使用することはできませんので注意してください。

何か質問やコメントがありましたら、Twitter（@fgbjr）またはSIST-02 GitHub issue tracker英語または日本語で気軽にお寄せください。
